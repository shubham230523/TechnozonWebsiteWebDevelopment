# Technozon-Website 🚀🌐

A comprehensive, multi-page technical event website developed for **"Technozon"**, the flagship technical festival of the New Horizon Institute of Technology and Management. This project serves as a digital hub for event information, registration, and participant engagement.

---

## 🎯 Project Overview

The Technozon website is designed to handle high volumes of student traffic and provides a seamless user journey through:
* **Event Discovery:** Browsing various technical competitions and workshops.
* **In-depth Details:** Detailed descriptions, rules, and schedules for each event.
* **Participant Registration:** A secure, database-driven registration system for event sign-ups.

---

## 🌟 Key Features

* **Responsive Design:** A mobile-friendly interface built with **Bootstrap**, ensuring accessibility across all devices.
* **Dynamic Event Pages:** Interactive event cards and detail pages powered by **jQuery**.
* **Database Integration:** A robust backend using **MySQL** to store and manage registration data.
* **Form Validation:** Client-side and server-side validation to ensure clean data entry.
* **Server-side Processing:** **PHP** scripts to handle form submissions and database queries.

## 🚀 Setup & Installation

1.  **Server Setup:** Install [XAMPP](https://www.apachefriends.org/) and ensure Apache and MySQL are running.
2.  **Clone the repository:**
    ```bash
    git clone [https://github.com/shubham230523/TechnozonWebsiteWebDevelopment.git](https://github.com/shubham230523/TechnozonWebsiteWebDevelopment.git)
    ```
3.  **Local Deployment:** Move the project folder into your XAMPP `htdocs` directory.
4.  **Database Configuration:**
    * Import the provided SQL schema into **phpMyAdmin**.
    * Update database credentials in your PHP connection script.
5.  **Access the Site:** Navigate to `http://localhost/TechnozonWebsiteWebDevelopment/` in your browser.
